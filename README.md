## 使用

在命令行中，进入到项目文件夹

运行`install_requirements.py`文件：

```bash
python install_requirements.py
```

运行`pyqt5_momotranslator_labelplus.py`文件：

```bash
python pyqt5_momotranslator_labelplus.py
```

运行程序后，将出现一个图形用户界面。
